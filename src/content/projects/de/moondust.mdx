MOONDUST war unsere dritte Rakete. Sie wurde im August 2025 in der Nähe von Bremen gestartet und erreichte die Zielhöhe von 1200 m. Obwohl das Bergungssystem eine verfrühte Auslösung des Hauptfallschirms erlebte, landete die Rakete sicher auf einem nahegelegenen Feld und wurde mit nur geringen Schäden geborgen. Die Mission war ein Erfolg und alle anderen Systeme funktionierten wie vorgesehen, was wertvolle Daten für die Entwicklung unserer nächsten Rakete, STARGAZE, bei der EuRoC 2025 lieferte.

Das Ziel von MOONDUST war es, neue Mitglieder einzuarbeiten, indem sie eine komplette Rakete von Grund auf basierend auf dem bestehenden STARDUST-Design bauten. Sie umfasste mehrere Neugestaltungen, insbesondere des Bergungssystems, um deren Funktionalität für den Einsatz in STARGAZE zu verifizieren.

| Daten         |           |
| ------------- | --------- |
| Höhe          | 1165 m    |
| Masse         | 12,2 kg   |
| Gesamtimpuls  | 2,869 kNs |
| Geschwindigkeit | 0,53 Ma |
