## Projektüberblick

Das CubeSat-Tracking-System ist eine umfassende webbasierte Plattform zur Echtzeit-Überwachung und Verfolgung der CubeSat-Missionen von BEARS. Dieses Projekt schließt die Lücke zwischen Bodenstationen und Missionskontrolle und bietet eine intuitive Benutzeroberfläche zur Visualisierung von Orbit- und Telemetriedaten.

### Hauptfunktionen

- **Echtzeit-Orbitpositionsverfolgung**: Live-3D-Visualisierung von Satellitenpositionen und -flugbahnen
- **Telemetrie-Dashboard**: Überwachung kritischer Systeme einschließlich Energieversorgung, Temperatur und Lageregelung
- **Bodenstationsintegration**: Nahtlose Anbindung an mehrere Bodenstationsnetzwerke
- **Historische Datenanalyse**: Abfrage und Analyse vergangener Missionsdaten für Forschungszwecke
- **Alarmsystem**: Automatische Benachrichtigungen bei Anomalien und kritischen Ereignissen

### Technischer Stack

- **Frontend**: React mit Three.js für 3D-Orbitalvisualisierung
- **Backend**: Node.js mit Express für API-Services
- **Datenbank**: PostgreSQL für Telemetriedatenspeicherung
- **Echtzeit-Kommunikation**: WebSocket-Verbindungen für Live-Datenstreaming
- **Deployment**: Docker-Container auf AWS-Infrastruktur

### Aktueller Status

Das System ist derzeit im Produktivbetrieb und verfolgt zwei aktive CubeSat-Missionen. Wir verbessern die Plattform kontinuierlich basierend auf dem Feedback der Missionsoperatoren und fügen neue Funktionen für kommende Starts hinzu.

### Zukunftspläne

- Integration zusätzlicher Bodenstationsnetzwerke
- Machine-Learning-basierte Anomalieerkennung
- Mobile App für die Überwachung von unterwegs
- Öffentliche API für Bildungseinrichtungen
